This repository contains a collection of cybersecurity tools and scripts developed to demonstrate practical security concepts and hands-on experience in cybersecurity.

The goal of this repository is to build and showcase security-focused projects that cover areas such as network security, authentication, vulnerability detection, and security automation.

These projects are developed primarily using Python and are intended for educational and research purposes.

---

## 📂 Current Projects

### 1️⃣ Network Port Scanner
A simple network port scanner that scans a target host to identify open ports and the services running on them.

**Features**
- Scans a range of ports
- Detects open ports on a target host
- Identifies common services running on ports
- Demonstrates basic network reconnaissance techniques

**Technologies Used**
- Python
- Socket Programming

---

### 2️⃣ Password Strength Checker
A tool that evaluates the strength of a password based on security best practices.

**Features**
- Checks password complexity
- Detects weak passwords
- Encourages strong password practices
- Demonstrates authentication security concepts

**Technologies Used**
- Python
- Regular Expressions
